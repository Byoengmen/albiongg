# 알비온 온라인 한국어 정보 사이트

알비온 온라인 플레이어를 위한 한국어 정보 및 계산기 모음 사이트입니다.  
GitHub Pages를 통해 배포되며, 모든 기능은 단일 HTML 파일로 제공됩니다.

**사이트 주소:** https://[your-username].github.io/Aibiongg

---

## 제공 기능

### 페임 보너스 계산기 (`index.html`)
필드사냥 기준 최종 페임 보너스를 계산합니다.

**입력 항목**
- **통찰력의 가방** — 티어(T4~T8) / 인첸트(.0~.4) / 품질(일반~걸작)
- **존 타입** — 블루존(+115%) / 옐로존(+140%) / 레드존(+230%) / 블랙존(+320%)
- **프리미엄** — 없음(×1.0) / 있음(×1.5)

**계산 공식**
```
최종 페임 보너스 = (존 보너스 + 가방 보너스) × 프리미엄 배수
```

---

## GitHub Pages 배포 방법

1. 이 저장소를 GitHub에 push
2. 저장소 **Settings → Pages** 이동
3. Source: `Deploy from a branch` 선택
4. Branch: `main` / `(root)` 선택 후 저장
5. 잠시 후 `https://[your-username].github.io/[repo-name]` 에서 접속 가능
